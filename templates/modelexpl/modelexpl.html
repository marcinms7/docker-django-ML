Neural Network Algorithm

My algorithm is programmed from scratch in Python. It is constructed from the initial hidden layer, specified amount of hidden layers and an output layer. It creates randomised weights, that then are updated with the learning rate being 0.01.



Each iteration goes through forward and back propagations.

Forward propagation

Here the prediction of outputs is calculated. In the code that starts with matrix or all hidden layers. Data propagates from the input layer, throughout the output layer.

Each neuron’s output is a logistic unit with Sigmoid activation function, where x is dot product of features and weights, plus arbitrary bias.

Backward propagation

Here, the ‘reversed’ iterator is used to back-propagate through neural network. Every time error is updated and appended to error list. Hidden neuron number j is index of the neuron’s weights. Error signal is computed with the derivative of our sigmoid function.

The process is repeated until convergence, using gradient descent algorithm. Standard learning rate for this model is currently 0.01

Here the prediction of outputs is calculated. In the code that starts with matrix or all hidden layers. Data propagates from the input layer, throughout the output layer.

Gradient descent can only find local minima/maxima, because the current form is not possible to analyse analytically (same as we cannot differentiate logistic regression, because of its Maximum Likelihood error finding solution). Each iteration of gradient descent updates the weights by delta time learning rate. Delta here is constructed of product of error and derivative of our neuron’s output function, in the literature this is computed often with the chain rule (see example below) but for simplicity I included another function (hower your mouse over that screenshot to display)
